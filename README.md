Download our project:

```
git clone https://github.com/Issachu/EZLabelTool.git
```

run front-end in our project

```
cd EZlabel/front-end

npm install

npm run start
```

run back-end in our project

```
cd EZlabel/back-end

pip install -r requirements.txt

flask run
```

Push to github
