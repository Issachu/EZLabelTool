Download our project:

```
git clone https://github.cs.adelaide.edu.au/MCI-Projects-2021/Team-01.git
```

run front-end in our project

```
cd EZlabel/front-end

npm install

npm run start
```

run back-end in our project

```
cd EZlabel/back-end

pip install -r requirements.txt

flask run
```
